# 🚨 Smart Smoke Detector System (Arduino)

مشروع نظام كاشف دخان ذكي يعتمد على منصة **Arduino** ومستشعر **MQ-2** لتوفير أمان وحماية من الحرائق وتسريب الغاز.

---

## 🛠 المكونات المستخدمة (Hardware)
* **Arduino Uno/Nano** (العقل المدبر).
* **MQ-2 Gas/Smoke Sensor** (للكشف عن الدخان والغاز).
* **Buzzer** (لإصدار صوت إنذار).
* **LED Red/Green** (لمعرفة حالة الجو الحالية).
* **Jumper Wires & Breadboard**.

---

## 🚀 المميزات
- [x] قياس مستمر لنسبة الدخان في الجو.
- [x] إنذار صوتي فوري عند تخطي نسبة الأمان.
- [x] تنبيه بصري باستخدام الـ LEDs.
- [x] كود سهل التعديل والتطوير.

---

## 💻 طريقة التشغيل
1. قم بتوصيل مستشعر **MQ-2** بالمدخل التماثلي (Analog Pin A0).
2. قم بتوصيل الـ **Buzzer** بالمدخل الرقمي (Digital Pin 10).
3. ارفع الكود الموجود في ملف `Smoke_Detector.ino` باستخدام **Arduino IDE**.
