# 🚨 مشروع كاشف الدخان الذكي (Smart Smoke Detector)

هذا المشروع ليس مجرد كاشف دخان، بل هو نظام متكامل مصمم للعمل بدقة عالية باستخدام تقنيات الـ IoT.

---

## 💡 فكرة المشروع
يهدف هذا النظام إلى تعزيز أمان البيئة المحيطة عبر استشعار وجود الدخان في الغرفة، وإطلاق إنذار فوري بمجرد تجاوز تركيز الدخان للحد المسموح به.

## 🛠️ المكونات التقنية
- **المتحكم:** Arduino Uno.
- **الحساس:** MQ-2 Smoke Sensor.
- **المخرجات:** Buzzer (للإنذار الصوتي) + LED (للإنذار الضوئي).

## 🚀 المميزات الرئيسية
- [x] **استجابة فورية:** الكشف عن الدخان في أقل من ثانية.
- [x] **إنذار مزدوج:** صوتي وضوئي لضمان التنبيه.
- [x] **كود مُحسن:** تم كتابة الكود ليستهلك أقل قدر ممكن من الذاكرة (Memory Optimized).

## ⚙️ كيف يعمل؟ (Logic Flow)
1. يقوم الحساس (MQ-2) بقراءة قيمة الجهد الناتجة عن وجود جزيئات الدخان.
2. يتم تحويل الإشارة التناظرية (Analog) إلى قيمة رقمية عبر الـ Arduino.
3. إذا تجاوزت القيمة حد العتبة (Threshold)، يتم تفعيل الـ Buzzer فوراً.

## 💻 الكود البرمجي (Snippet)
```cpp
// جزء من منطق الكشف
if (smokeValue > threshold) {
  digitalWrite(buzzerPin, HIGH);
  digitalWrite(ledPin, HIGH);
}
