# 🚨 مشروعSmart-Smoke-Detector  (Smart Smoke Detector)

هذا المشروع ليس مجرد كاشف دخان، بل هو نظام مصمم للعمل بدقة عالية باستخدام تقنيات الـ **IoT** لتعزيز الأمان في البيئة المحيطة.

---

## 🛠 المكونات التقنية (Hardware)
* **Arduino Uno:** العقل المدبر للمشروع.
* **MQ-2 Sensor:** حساس الدخان والغازات.
* **Buzzer:** جرس التنبيه الصوتي.
* **Jumper Wires:** أسلاك لتوصيل الدائرة.

---

## 🔌 مخطط التوصيل (Wiring Guide)
لضمان عمل الكود بشكل صحيح، يرجى التوصيل كالتالي:

| المكون (Component) | طرف الأردوينو (Arduino Pin) |
| :--- | :--- |
| **MQ-2 Sensor (VCC)** | 5V |
| **MQ-2 Sensor (GND)** | GND |
| **MQ-2 Sensor (AO)** | A0 |
| **Buzzer (+)** | Pin 13 |
| **Buzzer (-)** | GND |

---

## 🚀 كيفية التشغيل
1. قم بتوصيل الأردوينو بجهاز الكمبيوتر.
2. استخدم **Arduino IDE** لرفع ملف `main.ino`.
3. افتح الـ **Serial Monitor** لمراقبة قراءات الدخان في الوقت الفعلي.
