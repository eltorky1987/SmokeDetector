# 🔥 نظام كاشف الدخان الذكي | Smoke Detector IoT 🛡️

> **مشروع متكامل باستخدام Arduino وحساس MQ-2 للكشف عن الغاز والدخان وتنبيه المستخدم فوراً.**

---

### 📟 المكونات المستخدمة (Hardware)
* **Microcontroller:** Arduino Uno / Nano.
* **Sensor:** MQ-2 Gas & Smoke Sensor.
* **Alert System:** Buzzer (صفارة إنذار) + LED (إضاءة تحذيرية).
* **Power:** 5V External Power.

---

### 💻 الكود البرمجي (Software)
المشروع مكتوب بلغة **C++ (Arduino)** ويحتوي على:
* معايرة تلقائية للحساس عند التشغيل.
* نظام تنبيه صوتي ومرئي عند تجاوز نسبة الغاز للحد المسموح.

---

### 🛠️ طريقة التوصيل
* **A0:** موصل بالطرف التماثلي للحساس.
* **Pin 13:** موصل بالـ LED والـ Buzzer.
